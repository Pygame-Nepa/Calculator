# Calculator

A clean, responsive, and user-friendly online calculator built using web technology ,Made by:Arpan Poudel.

You can try live by onlinecalculatorbyarpan.netlify.app

# Features

Basic Arithmetic: Supports addition, subtraction, multiplication, and division.
Responsive UI: A clean, modern interface that works great on both desktop and mobile screens.
Instant Feedback: Features a live display area for real-time calculations.



# Built With

HTML5: For the structural layout of the calculator keys and display.
CSS3: For modern styling, button animations, and responsiveness.
JavaScript : For the calculation logic and button click events.
